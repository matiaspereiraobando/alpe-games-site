# Sprint reminders (Workflow Builder, `#alpe-game-lab`)

Create four scheduled messages per cycle. Easiest path: one Workflow per reminder,
each on a one-time schedule that you reset at the start of each cycle.

## Day 0 — kickoff

```
:rocket: *Cycle kickoff*
Lock scope: max 5 bullets in the game README.
Drop your pitch in this thread.
Workflow: https://github.com/matiaspereiraobando/alpe-games-site/blob/main/WORKFLOW.md
Checklist: https://github.com/matiaspereiraobando/alpe-games-site/blob/main/CYCLE-1.md
```

## Day 5 — mid-cycle check

```
:hourglass_flowing_sand: *Mid-cycle check*
Is the core loop playable end-to-end?
If no, cut scope NOW — easier today than on day 9.
```

## Day 9 — ruthless cut

```
:scissors: *Ruthless cut day*
Feature freeze. Only critical fixes from here.
Question: can a stranger play one full session without explanations we didn't build?
```

## Day 12 — ship checklist

```
:ship: *Ship checklist time*
- npm run build green
- Game live at /games/{slug}/
- Registry PR open on alpe-games-site
- Postmortem drafted
- Slack release post in #alpe-game-lab
```

## Cadence

The default cadence is 13–14 days per cycle. Adjust the four schedules at the
start of each cycle. If you slip a day, slip all four together.
